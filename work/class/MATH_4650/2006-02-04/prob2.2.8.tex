$g(x) = 2^{-x}$ has a fixed point because it is continuous and $g([1/3,1]) =
[1/\sqrt[3]{2}, 1/2] \subseteq [1/3,1]$.  In addition, $|g(x)| < 1$ for all $x$
in that interval, so the fixed point is unique.
